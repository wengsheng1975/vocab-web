# EnglishReader - 英语阅读学习网站

一个基于文章阅读的英语学习平台。通过导入英文文章、交互式阅读标记生词、智能生词库管理和水平追踪，帮助你系统地提升英语能力。

## 核心功能

1. **文章导入与难度评估** — 粘贴英文文章，系统自动分析难度等级（A1-C2），并判断是否适合你当前水平
2. **交互式阅读** — 阅读文章时，点击不认识的单词将其标红，系统自动记录
3. **智能生词库** — 跨文章合并生词，按词频排序（高频生词优先），上下文释义
4. **掌握度追踪** — 连续3次阅读中未点击的生词自动标记为已掌握；再次点击则重新加入
5. **阅读报告** — 每次阅读完成后生成统计报告：新增生词、高频提醒、掌握进度
6. **水平评估** — 系统根据阅读历史动态评估你的英语水平，并生成趋势图

## 技术栈

- 后端: Node.js + Express + SQLite (better-sqlite3)
- 前端: React 18 + Vite + React Router
- 认证: JWT

## 快速开始

```bash
# 终端1：启动后端
cd server
npm install
npm start
# 服务器运行在 http://localhost:3000

# 终端2：启动前端
cd client
npm install
npm run dev
# 前端运行在 http://localhost:5173
```

打开浏览器访问 http://localhost:5173，注册账号后即可开始使用。

## 使用流程

1. 注册/登录账号
2. 点击"导入文章"，粘贴一篇英文文章
3. 查看文章难度分析，点击"开始阅读"
4. 阅读过程中，点击不认识的单词（变红）
5. 阅读完毕后点击"完成阅读"，可为生词填写释义
6. 查看阅读报告，了解本次学习情况
7. 在"生词本"中复习和管理所有生词
8. 在"学习进度"中查看水平变化趋势
